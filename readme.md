untuk menjalankan aplikasi ini manual tanpa docker.

silahkan buat .env di masing masing root project frontend & backend

setelah di isi execute command ini di terminal root folder

---

backend development :

- npm install
- npm run dev

frontend development :

- npm install
- npm run dev

---

backend production :

- npm install
- npm run build
- npm run start

frontend production :

- npm install
- npm run build
- npm run preview
